# 项目名称：go-todo

## 项目介绍
go-todolist 是一个使用 Go 语言编写的简单的待办事项应用程序。它使用 Gin 框架和 GORM ORM 库。

## 安装
1. 克隆此存储库
2. 进入存储库目录
3. 运行 `go mod tidy` 命令以安装所有依赖项
4. 运行 `go run main.go` 命令以启动应用程序

## 使用
1. 打开浏览器并访问 `http://localhost:8080`
2. 单击“添加任务”按钮以添加新任务
3. 单击任务旁边的“完成”按钮以将任务标记为已完成
4. 单击任务旁边的“删除”按钮以删除任务
